Thông tin về trò chơi

Rambo3 là một trò chơi hành động bắn súng được phát triển để đáp ứng yêu cầu của môn học phát triển phần mềm mã nguồn mở. Game được xây dựng trên nền tảng Pygame và sử dụng ngôn ngữ Python. Trò chơi được thiết kế để chơi tối đa 2 người trên cùng một mạng LAN, giúp người chơi có thể tương tác với nhau và cùng trải nghiệm trò chơi. Trong game, người chơi sẽ vào vai nhân vật Rambo và chiến đấu với người chơi còn lại. Để tạo ra trò chơi này, mình đã sử dụng kiến thức về socket và thread để xây dựng tính năng chơi game đa người chơi. Mình cũng đã áp dụng các kỹ thuật lập trình hướng đối tượng để tạo ra các đối tượng trong game như vũ khí, bậc đá và nhân vật Rambo. Nếu bạn là một người yêu thích trò chơi hành động và muốn trải nghiệm trò chơi Rambo3 cùng với bạn bè, hãy tải game và tham gia vào cuộc chiến ngay bây giờ!

Điều khiển nhân vật
  • Di chuyển: Sử dụng các phím mũi tên trên bàn phím để di chuyển
  • Bắn súng: Ấn phím "s" trên bàn phím để bắn
  • Số đạn tối đa: Mỗi người chơi chỉ có tối đa 2 viên đạn trên cửa sổ trò chơi, nếu muốn bắn tiếp phải chờ cho đạn trúng đích, hoặc đạn bay ra ngoài
  • HP: Mỗi người chơi có 5 HP ( trúng đạn tối đa 5 lần )
  • Chướng ngại vật: Mỗi bên của người chơi có một mỏm đá, người chơi có thể nhảy lên để né đạn

Các gói thư viện cần thiết để chạy trò chơi
Trên Linux: Python3, Pygame
Cài đặt python3 trên Linux
  • Bước 1: Mở Terminal
  • Bước 2: Bắt đầu cài đặt
    sudo apt update
    sudo apt install python3
  • Bước 3: Kiểm tra cài đặt Sau khi cài đặt hoàn tất, bạn có thể kiểm tra phiên bản Python 3 bằng cách nhập lệnh sau trong Terminal:
    python3 --version
    Nếu Python đã được cài đặt thành công, phiên bản Python và thông tin liên quan sẽ được hiển thị trên màn hình.
Cài đặt Pygame trên Linux
  • Bước 1: Cài đặt pip nếu chưa có sudo apt update
    sudo apt install python3-pip
  • Bước 2: Cài đặt các phụ thuộc cần thiết của pygame
    sudo apt install python3-dev python3-pygame
  • Bước 3: Cài đặt pygame
    sudo pip3 install pygame
  • Bước 4: Kiểm tra cài đặt. Mở Terminal nhập lệnh sau:
    python -m pygame.examples.aliens
    Nếu một trò chơi Aliens đang chạy, điều này có nghĩa là Pygame đã được cài đặt thành công.
Trên Window: Python3, Pygame
Cài đặt Python3 trên Window
  • Bước 1: Tải xuống trình cài đặt Truy cập trang web chính thức của Python (https://www.python.org/downloads/) và tải xuống phiên bản mới nhất của Python 3. Nhấp đúp vào tệp tin cài đặt đã tải xuống để bắt đầu quá trình cài đặt.
  • Bước 2: Bắt đầu cài đặt Chọn "Install Now" để cài đặt Python 3 với các tùy chọn mặc định. Nếu bạn muốn thay đổi cài đặt hoặc cài đặt Python 3 cho một người dùng khác, hãy chọn "Customize installation".
  • Bước 3: Chờ đợi quá trình cài đặt hoàn tất Chờ đợi quá trình cài đặt hoàn tất, điều này có thể mất vài phút tùy thuộc vào tốc độ máy tính của bạn.
  • Bước 4: Kiểm tra cài đặt Mở cửa sổ Command Prompt bằng cách bấm phím "Win + R" sau đó gõ "cmd" và nhấn Enter. Nhập lệnh "python" để kiểm tra xem Python đã được cài đặt thành công chưa. Nếu Python đã được cài đặt thành công, phiên bản Python và thông tin liên quan sẽ được hiển thị trên màn hình.

    python -m pygame.examples.aliens
    Nếu một trò chơi Aliens đang chạy, điều này có nghĩa là Pygame đã được cài đặt thành công.
